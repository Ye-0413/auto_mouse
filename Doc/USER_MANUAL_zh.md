# Anything Auto 用户手册

本手册说明桌面客户端的日常使用方式。安装命令行、依赖与打包请参阅仓库内 `docs/RUN.md`、`docs/BUILD.md`。

---

## 1. 产品与适用场景

Anything Auto 是一款**在本机运行**的轻量 RPA 客户端：用可视化流程描述「点哪里、输什么、等什么」，按顺序执行桌面与（可选）浏览器操作。

**推荐使用方式：**

1. 在 **「流程」** 页用画布编排步骤并保存。  
2. 在 **「配置」** 中设置默认流程、超时、截图、浏览器 CDP 等。  
3. 在 **「运行」** 选择流程并单次执行。  
4. 在 **「日志」** 核对执行结果与每一步明细。

**环境与数据**

- **Windows** 上对窗口激活、快捷键、坐标点击等行为验证最充分；其他系统亦可运行，请以实际界面为准。  
- 流程、配置、执行记录保存在本机 **SQLite**。  
- 可通过环境变量 **`ANYTHING_AUTO_DATA_DIR`** 指定数据目录；主窗口底部状态栏会显示当前数据库路径。

## 2. 安装与启动

```bash
pip install -r requirements.txt
python main.py
```

可选能力（按需安装）：

| 能力 | 说明 |
|------|------|
| 录制 | `pip install pynput` 或 `pip install -e ".[recorder]"` |
| 浏览器步骤（Playwright / CDP） | `pip install -e ".[browser]"`，并按官方说明安装 Chromium |

---

## 3. 主界面导航

左侧为分区列表，自上而下为：

| 分区 | 作用 |
|------|------|
| **运行** | 选择一个流程，**单次执行**；底部显示本次输出摘要。 |
| **流程** | 画布编排步骤，新建/删除流程，保存定义。 |
| **录制** | 录制鼠标点击、键盘输入与组合键，保存为新流程或追加到已有流程。 |
| **配置** | 多个配置档案；默认流程、超时、重试、失败截图、窗口/CDP 等。 |
| **日志** | 最近执行列表；双击可查看步骤级明细；支持导出 CSV。 |

切换分区时，部分页面会把键盘焦点移动到该页常用控件（便于键盘与辅助技术使用）。

## 4. 流程页：画布编排

流程的**执行顺序完全由画布上的从左到右顺序决定**，与「拖拽后的横向次序」一致，而不是单独的表格排序。

### 4.1 鼠标操作

- **拖拽卡片**：横向调整位置；静置约 **0.16 秒** 后会按 **从左到右** 对齐连线并刷新顺序（与画布提示一致）。  
- **双击卡片**：打开步骤编辑对话框，修改参数。  
- **框选**：与 Qt 画布默认行为一致。  
- **Ctrl + 滚轮**：缩放画布。

### 4.2 顶部工具栏

- **流程下拉框**：切换正在编辑的流程。  
- **新建流程 / 保存 / 删除流程 / 刷新列表**：管理流程记录。删除流程前会二次确认（含步骤数量提示）。  
- **流程名称、状态**：与保存内容一并写入数据库。  
- **添加步骤**：打开对话框新增一步，并出现在画布上。  
- **编辑 / 删除**：针对**画布上当前选中的一张卡片**；删除步骤前会确认（不可逆，需再点保存才会写库）。

### 4.3 汇入节点

点击画布区域内 **「添加汇入节点」** 可在链路中插入菱形 **汇入** 占位（执行时常以「备注/跳过」类步骤处理，具体以引擎为准）。位置会写入流程定义中的画布布局数据。

### 4.4 画布键盘（焦点在画布视图时）

请先 **用 Tab 键将焦点移到流程画布区域**（或先用鼠标点一下画布空白处），再使用：

| 按键 | 作用 |
|------|------|
| **← / →** | 在步骤卡片之间切换选中。 |
| **Ctrl + Shift + ← / →** | 与**相邻**卡片交换执行顺序；交换后会按当前顺序自动规整横向坐标，避免连线错乱。 |
| **Enter** | 编辑当前选中的步骤。 |
| **Delete / Backspace** | 删除当前选中步骤（与工具栏删除相同确认框）。 |

未选中任何卡片时，方向键不会改变选中项；请先单击一张卡片或使用 Tab 聚焦画布后再操作。

### 4.5 变量替换

步骤参数（如文本、路径、网址等）支持 **`${变量名}`**。单次运行时由流程上下文赋值；常与「读取剪贴板」「设置变量」「条件分支」等配合使用。

### 4.6 保存

点击 **保存** 前，应用会冲刷画布坐标并与步骤对齐；若 DSL 校验失败会提示错误列表，数据库不会写入不完整定义。

## 5. 运行页（单次执行）

- 下拉选择要跑的流程（可与配置中的默认流程联动）。  
- 点击 **运行**；执行过程中可看页面内输出区域。  
- 若某项步骤失败且配置中开启了 **失败时截图**，可能生成截图路径（详情请查日志或配置说明）。

## 6. 步骤类型一览

以下为常见步骤类型（名称以界面标签为准）：

| 类别 | 说明 |
|------|------|
| 等待 | 暂停指定毫秒。 |
| 输入文字 | 向当前键盘焦点处输入文本。 |
| 快捷键 | 组合键（如 Ctrl+V）。 |
| 激活窗口 | 按窗口标题片段激活窗口。 |
| 打开网址 / 打开文件 | 系统浏览器或默认程序打开。 |
| 鼠标点击 | 屏幕坐标点击（分辨率变化时需重调）。 |
| 粘贴剪贴板 | 向当前焦点控件粘贴剪贴板内容。 |
| 读取剪贴板 | 读到命名变量（默认可与分支配合）。 |
| 清空剪贴板 | 将系统剪贴板置空。 |
| 剪贴板关键词分支 (`clipboard_switch`) | 根据变量内容与 JSON 规则匹配分支；无任何匹配时**整条流程静默结束**（不报错）。 |
| 条件分支 (`if`) | JSON 数组描述 then/else 子步骤序列。 |
| 设置变量 | 运行期写入上下文变量。 |
| 备注 / 汇入占位 | 说明或画布装饰，一般不产生自动化动作。 |
| Playwright 相关 (`pw_*`) | 需安装浏览器可选依赖并配置 CDP。 |

### 剪贴板分支（简要）

1. 通常先用 **读取剪贴板** 写入变量（如默认 `_clipboard`）。  
2. **剪贴板关键词分支** 中按规则的 `contains_any` 等字段**顺序**匹配；**先命中先执行**该规则内的 `steps`。  
3. 若所有规则均未命中：**流程静默结束**，后续步骤不再执行。

在步骤编辑器中，`rules` 的 **JSON** 下方会显示**实时语法与结构提示**（保存时仍会逐项校验）。

### 条件分支（简要）

- `then`、`else` 为 **JSON 数组**，元素为与子流程相同的步骤对象。  
- 编辑器中会对两段 JSON **实时提示**是否可被解析为数组；确定前请确认比较符与左右操作数字段填写正确。

## 7. 录制页

依赖 **pynput**：

```bash
pip install -e ".[recorder]"
# 或
pip install pynput
```

**全局快捷键（应用内注册）：**

| 快捷键 | 作用 |
|--------|------|
| **Ctrl + Shift + R** | 开始录制 |
| **Ctrl + Shift + S** | 停止录制 |

录制进行中也常可用 **F12** 立即停止（无需切回本窗口）。

录制结果可 **保存为新流程** 或 **追加到所选流程**，保存前会做 DSL 校验。

---

## 8. 配置页

可维护多份配置档案：

- **使用的流程**：与运行页默认值等保持一致性。  
- **目标窗口标题 / 浏览器标题 / browser_cdp_url**：供桌面与浏览器步骤使用。  
- **默认超时、重试次数**。  
- **失败时截图** 开关。

修改后务必点击 **保存**。删除档案至少需要保留一份配置（界面会提示）。

---

## 9. 日志页

- 表格展示近期执行：**状态、流程 ID、时间与说明**等。  
- **双击一行**打开该次执行的**步骤明细**对话框。  
- **导出 CSV…**：导出当前列表范围的记录（编码含 BOM，便于 Excel 打开）。

---

## 10. 无障碍与键盘提示

- 侧栏分页切换后，界面会尽力将焦点放到该页的常用控件。  
- 画布区域已为视图设置可访问名称与快捷键说明（读屏软件是否朗读取决于操作系统与 Qt/辅助技术组合）。  
- 步骤编辑器内，剪贴板分支与条件分支的 **JSON 状态**行带有可读名称，便于辨认校验结果。

若读屏与实际焦点不一致，可尝试先用 **鼠标单击**对应区域再 Tab 遍历。

## 11. 常见问题

| 现象 | 建议 |
|------|------|
| 浏览器步骤执行失败 | 确认已安装 `anything-auto[browser]`、`playwright install chromium`，且 CDP 地址与浏览器远端调试一致。 |
| 按键或文本未进目标控件 | 先 **激活窗口** 并把焦点点到输入框，再执行输入/粘贴步骤。 |
| 坐标点击不准 | 更换分辨率或多显示器后坐标易偏；改用窗口/文本/浏览器定位更安全。 |
| 剪贴板分支「突然结束」 | 属于**无规则命中**的设计行为；检查变量是否先被读取、`contains_any` 与大小写选项是否匹配内容。 |
| 找不到数据库 | 看状态栏路径；或设置 `ANYTHING_AUTO_DATA_DIR` 后重启。 |
